副标题：一个只做中英的浏览器扩展，划词、整页、视频同传一次搞定

（此处插入图片：icon-128.png，居中）

刷英文资料，最耗神的不是单词，是来回切。

一段话看不懂 → Ctrl+C → 打开翻译网站 → Ctrl+V → 看完再切回去。
视频更狠：字幕是英文，耳朵跟不上，手还得去找别的工具。

有没有一种可能：人还在这个页面上，翻译就已经发生了？

有。它叫 Lingua Bridge。

它解决什么

一句话：
中英互译，嵌进你正在看的网页和视频里。

不用复制。不用新开十个标签。不用先纠结「要不要注册、要不要先买 Key」。

打开扩展主开关，事情就变简单了。

① 划词：选中就懂

鼠标一拖，气泡出来。点翻译，原文译文对照。
还能看词条讲解——音标、释义。适合边读边记。

（此处插入图片：selection-bubble.png）

不用离开页面，上下文还在。这就是「好用」的第一感。

② 控制台：一眼知道开没开

主开关、供应商、同传样式、本页有没有视频、同传是否在跑——弹窗里写得清清楚楚。

（此处插入图片：popup.png）

你可以选：

· 静默字幕：只看双语浮层，不吵
· 语音传译：听译文，同时还能看字幕

两种模式，按场景切换就行。

③ 视频同传：字幕浮在画面上

看英文演讲、技术分享时，开启同传，双语字幕直接叠在视频上——可拖动、可看历史、可折叠。

（此处插入图片：si-caption.png）

关字幕窗，同传也会停。状态干净，不拖泥带水。

④ 无 Key 也能用，有 Key 更强

设置页写得很直白：Works without a key · optional API key。

（此处插入图片：options.png）

· 没有 Key：先用起来
· 有 Key：接 OpenAI / DeepSeek / 讯飞 / OpenRouter… 讲解和视频同传更强
· Key 存在本机，可选加密保险库——没有自建后端，密钥不上传本项目服务器

为什么说「好用」而不只是「能用」

少操作
自动识别中英文，互译到另一侧。你不用每次选手动方向。

低门槛
没有 API Key 也能先体验；觉得值了再加 Key。

更安心
密钥留在你自己的浏览器里。

更聚焦
它只做中英。不是功能越少越好，而是：中英场景下更稳、设置更少。你要的是顺手，不是「全世界语言我都沾一点」。

和其他工具比，你省下的是什么

· 比百度/谷歌翻译网页版：省掉复制粘贴和切窗口
· 比浏览器自带整页翻译：多了划词讲解，还多了视频同传
· 比「什么语种都支持」的大插件：中英路径更短，干扰更少
· 比只查词的词典扩展：整页 + 视频同传也在同一个扩展里
· 比单独装同传软件：浏览器里直接开，和日常划词是同一套习惯

如果你的日常主要是中英，这种「窄而深」往往比「宽而浅」更省命。

谁会立刻爱上它

· 天天啃英文文档、技术博客的人
· 看 YouTube / 英文课，想要双语字幕甚至听中文的人
· 不想把网页内容一遍遍拷进翻译站的人
· 有自己的 API Key，又希望 Key 留在本机的人

反过来：如果你主要需要日韩或其他小语种，它不是你的菜——产品定位写得很清楚：Chinese ↔ English only。

怎么开始（真的很快）

1. 安装扩展（Chrome / Edge / Firefox）
2. 点开弹窗，打开主开关
3. 网页划词试试；视频页再点一次「开启同传」
4. 觉得好用了，再去设置页按需加 Key

三分钟能完成从「装上」到「第一次顺手」。

英文读不懂，可以查。
但流程太碎，会悄悄偷走你大量注意力。

Lingua Bridge 做的事很朴素：
让翻译发生在内容旁边，而不是发生在另一个标签页里。

只做中英。页面内完成。视频也能同传。无 Key 可起步，有 Key 可加强。
